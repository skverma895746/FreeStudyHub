# 📚 FreeStudyHub

FreeStudyHub is a **responsive frontend web project** built using HTML, CSS, and JavaScript.
It provides students with easy access to **study materials stored on Google Drive**.

---

## 🚀 Features

* 📂 Batch-wise study materials (9th, B.Tech, etc.)
* 🔗 Direct Google Drive links for files
* 📱 Fully responsive design (mobile + desktop)
* ⚡ Fast and lightweight UI
* 🎯 Simple and user-friendly interface

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Google Drive (for file storage)

## 🔄 How It Works

```id="flow2"

        ↓
Copy shareable link
        ↓
Paste link into frontend (HTML/JS)
        ↓
User clicks → File opens
```
